## 3. 동의 구조 (ISMS-P 2.3.1 필수·선택 분리)

```
전체 동의 (선택)
├── [필수] 서비스 이용 약관 동의
├── [필수] 개인정보 수집·이용 동의 (서비스 제공)
│         └── 이름, 이메일, 휴대전화, 생년월일, 비밀번호
├── [필수] 개인정보 수집·이용 동의 (결제·환불)
│         └── 결제수단 정보, 거래 내역
├── [필수] 개인정보 제3자 제공 동의 (결제 대행사)
│         └── 나이스페이(주) - 결제 처리
│
├── [선택] 개인정보 수집·이용 동의 (마케팅)
│         └── 이름, 이메일, 전화, 관심언어, 이용이력
└── [선택] 광고성 정보 수신 동의 (이메일·SMS·푸시)
```

> **ISMS-P 핵심 원칙:** 필수 동의와 선택 동의를 반드시 분리하여 각각 독립적으로 동의받아야 하며,
> 선택 동의 거부 시 서비스 이용에 불이익이 없어야 합니다.

| 법정대리인 동의 | 부모/후견인 정보 입력 + 인증 | 개인정보보호법 제22조 제6항 |

| 법정대리인 권리 보장 | 아동의 개인정보 열람·정정·삭제·처리정지 요청 가능 | 개인정보보호법 제38조 |
